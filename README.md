# TradeBuddy

**Unified messaging for tradespeople** — a Progressive Web App (PWA) built with React + Vite.

TradeBuddy brings Facebook Messenger, Facebook Page Comments, WhatsApp Business, and SMS into a single inbox so tradespeople can see and respond to all customer enquiries from one place.

## Features

- Unified Inbox with channel filter tabs
- Full message thread history per contact
- Quick Reply Templates (categorised, searchable)
- Voice Reply with AI professional reword
- Auto-Reply with optional working hours
- Business Profile personalisation
- PWA — installable on iPhone/Android home screen

## Development

pnpm install && pnpm dev

## Deploy to Railway

1. Push to GitHub
2. railway.app → New Project → Deploy from GitHub repo
3. Railway auto-detects nixpacks.toml and deploys
